# 📊 Business Performance Dashboard (SAP Analytics Cloud)

## 📌 Project Overview

This project showcases an end-to-end business analytics dashboard built using SAP Analytics Cloud (SAC). The dashboard provides insights into sales, profit, and product performance across different regions and time periods.

## 🎯 Objectives

* Analyze sales and profitability trends
* Identify high-performing and low-performing regions
* Evaluate product category performance
* Support business decision-making using data insights

## 📊 Key Features

* KPI metrics: Total Sales, Profit, Quantity
* Sales trend analysis over time
* Region-wise sales comparison
* Product category profitability analysis
* Scatter plot for Sales vs Profit analysis
* Interactive filters (Region, Category, Date)

## 🛠 Tools Used

* SAP Analytics Cloud (SAC)
* Data Visualization & Dashboarding
